With hiring processes becoming increasingly digital, I decided to explore how HR tools work behind the scenes by building a simple yet effective Applicant Tracking System (ATS). The project focuses on parsing resumes, extracting useful candidate data, and automating the filtering process based on job-specific keywords and qualification matching. I used Python for backend logic and Flask for setting up the web interface. The resume upload functionality supports PDF and DOCX files, and I used libraries like pyresparser and spacy to extract names, skills, education, and experience. These parsed details were stored in a structured format for analysis. A major part of the system was building smart filters: selecting candidates based on specific job roles, highlighting matching keywords, and tracking the status of each applicant. I added interactive filters like years of experience, skill matches, and education level to refine the shortlisting process. The UI featured real-time updates, candidate status tracking, and downloadable reports. This ATS project introduced me to how even simple automation tools can drastically reduce manual screening effort in recruitment.
